## Creating a leave channel

<Steps>
  <Step title="Head over to your Discord Server">
    Click on the Discord Server that you would like to setup leave for using
    greed.
  </Step>
  <Step title="Create the channel">
    Setup permissions for the channel with the desired name, for this example we
    will be using `#leave`
  </Step>
</Steps>

## Creating an embed and setting the leave channel

<Steps>
  <Step title="Creating the leave embed">
    Run the `,embedsetup` command while to create the leave embed which you
    can use variables with, for more information regarding variables:
    [/configuration/variables](/configuration/variables)
  </Step>
  <Step title="Setup the leave channel">
    Run the `,leave add` command which will give you further information, for this
    example we'll run: `,leave add #leave {title: Goodbye!} {author: {user.name}}`
  </Step>
</Steps>
