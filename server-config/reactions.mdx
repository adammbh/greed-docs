## Reaction Roles

Allow members to assign roles to themselves.

Subcommand: `reactionrole add` <br /> Module: `reactionroles` <br /> Description: Add a new reaction role to a message.

<div id="rr-add"/>
<Tabs>
  <Tab title="Example">
  `reactionrole add https://discord.com/channels/12475285...... :wave: @pic perms`
  <Frame caption="Adding a reaction role">
  <img src="/images/rr/1.gif" />
  </Frame>
  </Tab>
  <Tab title="Syntax">`reactionrole add <message> <emoji> <role>`</Tab>
  <Tab title="Flags">
    No flags are avaliable for this command.
  </Tab>
</Tabs>

<br />

Subcommand: `reactionrole remove` <br /> Module: `reactionroles` <br /> Description: Remove a reaction role from a message.

<div id="rr-remove"/>
<Tabs>

<div id="r-add"/>
<Tabs>
