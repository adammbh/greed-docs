## Base role

<Steps>
  <Step title="Head over to your Discord Server">
    Click on the Discord Server that you would like to setup Booster roles for using
    greed.
  </Step>
  <Step title="Create the base role">
  Use `,boosterrole base <role>` To Setup the base position for booster roles. For example `,boosterrole base @base` 
  <br />
  <Frame caption="Creating base role.">
  <img src="/images/booster/booster.gif" />
</Frame>
  </Step>
</Steps>

## Creating booster role
<Steps>
<Step title="Create the Booster Role">
  After setting up the base role, go to your commands channel and use `,br (color)`. For example, `,br #fff` sets the color. To rename your boost role, use `,br rename (name)`.
  <br />
  <Frame caption="Creating booster role.">
    <img src="/images/booster/booster1.gif" />
  </Frame>
</Step>
<Step>
  You can also add non-boosters and assign them their own booster role, allowing you to manage and recognize different groups within your server effectively.
  <br />
  <Frame caption="Allowing a role to bypass the booster requirement.">
    <img src="/images/booster/booster2.gif" />
  </Frame>
</Step>
</Steps>
