## Banning users

<Info>
  The variables used are for example only, to check the rest of the variables,
  visit [/configuration/variables](/configuration/variables)
</Info>

<Steps>
  <Step title="Head over to your Discord Server">
    Click on the Discord Server where you want to ban a user.
  </Step>
  <Step title="Banning the user">
    Use the bot command `,ban @username [reason]` in any text channel. Replace `@username` with the actual username of the user you want to ban and `[reason]` with the reason for the ban if necessary. The bot (greed) will then ban the user and notify you of the action.
  </Step>
</Steps>

## Giving roles

<Steps>
  <Step title="Adding roles">
    Use the bot command `,role add @username roleName` in any text channel. Replace `@username` with the actual username of the member you want to add the role to and `roleName` with the name of the role. For example, `,role add @johnDoe Moderator` will add the "Moderator" role to the user @johnDoe. The bot (greed) will then add the role and notify you of the action.
  </Step>
  <Step title="Managing roles for all or bots">
    You can also use the commands `,role all roleName` or `,role bots roleName` to add or remove a role for all members or all bots in the server. For example, `,role all Member` will add the "Member" role to all users, and `,role bots BotRole` will add the "BotRole" to all bots. The bot (greed) will then apply the role changes and notify you of the action.
  </Step>
</Steps>

## Removing roles

<Steps>
  <Step title="Removing roles">
    Use the bot command `,role remove @username roleName` in any text channel. Replace `@username` with the actual username of the member you want to remove the role from and `roleName` with the name of the role. For example, `,role remove @x6ur Moderator` will remove the "Moderator" role from the user @x6ur. The bot (greed) will then remove the role and notify you of the action.
  </Step>
</Steps>

## Changing role colors

<Steps>
  <Step title="Changing role color">
    Use the bot command `,role color roleName color` in any text channel. Replace `roleName` with the name of the role you want to change the color of and `color` with the desired color code or name. For example, `,role color Moderator #FF5733` will change the color of the "Moderator" role to the specified color. The bot (greed) will then update the role color and notify you of the action.
  </Step>
</Steps>

## Adding / Changing role icons 

<Steps>
  <Step title="Setting the role icon">
    Use the bot command `,role icon roleName icon|attachment|url` in any text channel. Replace `roleName` with the name of the role you want to set the icon for, and `icon|attachment|url` with the appropriate icon, attachment, or URL. For example, `,role icon Moderator :shield:` will set the shield emoji as the icon for the "Moderator" role, or `,role icon Moderator https://example.com/icon.png` will set the icon from the provided URL. The bot (greed) will then update the role icon and notify you of the action.
  </Step>
</Steps>

## Muting members / Timeout

<Steps>
  <Step title="Muting or Timing Out a User">
    Use the bot command `,timeout (user) (time)` in any text channel, for example to mute a user for 1 minute `,mute (x6ur) (1m)`. Both commands perform the same action, temporarily restricting the user's ability to send messages.
  </Step>
</Steps>

## Locking / Unlocking a channel

<Steps>
  <Step title="Locking a Channel">
    Use the bot command `,lock` in any text channel to lock the channel. This command will prevent users from sending messages in the channel until it is unlocked.
  </Step>
  <Step title="Unlocking a Channel">
    Use the bot command `,unlock` in any text channel to unlock the channel. This command will allow users to send messages in the channel again.
  </Step>
</Steps>
